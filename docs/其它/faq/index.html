    快速体验 #

    SuperSonic自带样例的语义模型和问答对话，只需以下三步即可快速体验：

    • 从release page下载预先构建好的发行包
    • 运行 “assembly/bin/supersonic-daemon.sh start"启动standalone模式的Java服务
    • 在浏览器访问http://localhost:9080 开启探索
